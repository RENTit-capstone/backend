API 기본 응답 형식

모든 API 응답은 아래와 같은 JSON 형식을 따릅니다:

{
  "success": "API 성공 여부(true, false)",
  "data": "<요청에 따라 달라짐>",
  "message": "성공시 빈 문자열, 실패시 에러 메시지"
}
